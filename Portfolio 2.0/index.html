My Game Projects

The Monster Pit

The Monster Pit is a web game project developed at RIT. The game centers around a single elvish archer, fighting relentlessly against hordes of spiders! Through the use of JavaScript and PixiJS, the player is able to move the character using the ASDW keys, aim/fire using the mouse, and collect a small variety of power-ups to prolong their survival!

Play the game: The Monster Pit (Computer use reccomended!)

Electrodungeon

Electrodungeon is a group board game developed at RIT. In this game, 2-4 players utilize a series of skill stats in order to traverse and attempt to escape a dizzying labyrinth powered by deadly electricity. However, escape won't come easy, as another player: The Master of Fates, will do everything in their power to prevent the players from escaping!

View the rulesheet: Electrodungeon Rules

Draconic Heights

Draconic Heights is a Unity game project developed at RIT. As enemies threaten to take control of the skies, it's your job to soar high and blast the threats out from the clouds. This game is simple "Shoot Em Up", where you take up the wings of a courageous dragon rider and earn as many points as you can defeating enemies! ASDW are used to move and space is used to shoot fireballs.

Play the game: Draconic Heights (Computer use reccomended!)

My Web Projects

Pokemon Web Finder

This simple Pokemon Finder is a web project developed at RIT. This webpage centers around using JavaScript and the Pokemon API to fetch data on a searched pokemon (if it exists) and format the information using HTML/CSS. Users are even able to filter for shiny, gender different and/or mega visuals, if they exist. Additionally, this application will always save the last search term after being closed.

View the site: Pokemon Finder
View the code: GitHub

Jedi Advice 101

Jedi Advice 101 is a smaller web project developed at RIT. This simple webpage focuses on utilizing JavaScript in order to hold an array of popular jedi quotes from various Star Wars sources and displays a random quote every time the button is clicked/tapped. The interface was created using a combination of HTML and CSS, in order to create a neatly formatted display for the user.

View the site: Jedi Advice 101
